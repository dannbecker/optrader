O Optrader é um website que utiliza HTML + CSS + Javascript, com Bootstrap para auxiliar na responsividade. O website está totalmente responsível, com todos os erros de layout corrigidos para visualização em Desktop Widescreen, Desktop 5:4, Tablets e Smartphones com diferentes níveis de resolução.

Possui 4 páginas no momento: Esta (o index), a de cálculo de porcentagem, a de cálculo de opções (do mercado de opções) e a guia 'sobre'.
